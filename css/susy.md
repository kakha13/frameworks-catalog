Susy was a responsive layout engine for Sass, before flexbox and CSS grid were available.

Susy is now deprecated, and will not receive updates. If you need help moving off Susy, or learning the latest in web layout, we offer training and consulting to help bring you up-to-date.

Learn about Susy https://www.oddbird.net/susy/
